Настоящая база данных представляет собой маленький фрагмент базы, создаваемой в рамках научно-исследовательноского проекта.

БД состоит из трёх таблиц:
1) text - содержит информацию о текстах научных работ (id текста, название, id автора, id научного руководителя, ключевые слова, год защиты);
2) students - содержит информацию о студентах (id студента, имя, факультет, год поступления);
3) teachers - содержит информацию о преподавателях (id преподавателя, имя, факультет, научня степень, читаемые курсы).

Таблицы являются лишь способом демонстрации БД в рамках учебного задания, поэтому не предендует на полноту и точность представленных сведений.
